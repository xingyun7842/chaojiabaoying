# 吵架包赢 🗣️

一个让你在任何争吵中都能获胜的神奇网站！

## 功能特色

- ✨ **智能回复生成** - 基于大模型AI生成高质量的反驳内容
- 🎛️ **语气强度调节** - 1-10级可调节的语气强烈程度
- 📱 **完美响应式** - 手机端和电脑端都有优秀的使用体验
- 💾 **历史记录** - 自动保存吵架历史，支持查看和删除
- 🎨 **微信风格设计** - 采用微信风格的配色和界面设计

## 技术栈

- **框架**: Next.js 14
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **存储**: localStorage（无需数据库）
- **AI模型**: GPT-3.5-turbo

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 运行开发服务器

```bash
npm run dev
```

### 3. 打开浏览器

访问 [http://localhost:3000](http://localhost:3000) 查看应用。

## 使用方法

1. **输入对方的话** - 在文本框中输入对方说的话
2. **调节语气强度** - 拖动滑块选择1-10的语气强烈程度
3. **点击"开始吵架"** - 获取3条神级回复
4. **查看历史记录** - 在页面底部可以查看历史吵架记录

```
├── src/

│   │   ├── globals.css      # 全局样式

│   │   └── page.tsx         # 主页面
│   ├── components/
│   │   └── History.tsx      # 历史记录组件
│   └── services/
│       ├── api.ts           # API调用服务
│       └── storage.ts       # localStorage服务

├── tsconfig.json           # TypeScript配置
└── package.json            # 项目配置
```

## API配置

项目使用了云雾AI的API服务，配置信息已集成在代码中。如需更改API配置，请修改 `src/services/api.ts` 文件。

## 注意事项

- 项目使用localStorage存储历史记录，清除浏览器数据会导致历史记录丢失
- API调用可能产生费用，请注意使用频率
- 建议在微信浏览器中打开以获得最佳体验
